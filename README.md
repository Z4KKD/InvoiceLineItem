# Invoice Line Item Calculator

This Python script calculates the total cost for a given quantity of items at a specified price per item. It prompts the user to enter the price and quantity for each line item, calculates the total cost, and displays the result.

## Example

```
The Invoice Line Item Calculator

Enter price: 10.5
Enter quantity: 3

PRICE:      10.50
QUANTITY:   3
TOTAL:      31.50
Enter another line item? (y/n): y

Enter price: 7.25
Enter quantity: 5

PRICE:      7.25
QUANTITY:   5
TOTAL:      36.25
Enter another line item? (y/n): n

Bye!
```
